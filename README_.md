## 功能

### 1. 登录助手

- 教务系统登录页、统一身份认证页自动填写账号密码
- 调用你配置的 OCR 服务识别验证码
- 可选：识别后自动点击登录

### 2. 评教助手

- 自动填写分数题 / 单选题 / 多选题 / 主观题
- 可配置：
  - 分数随机区间
  - 单选默认选项池、按题配置
  - 多选默认勾选池、按题配置
  - 主观题多行模板
- 可选「到时自动保存」：默认等待约 **100 秒** 后再提交  
  （教务侧有停留/倒计时校验，**不能**靠脚本真正跳过）

### 3. 全自动评教

在「教学评估」列表页启动后：

1. 扫描未评估问卷  
2. 逐个进入  
3. 自动填写  
4. 等待设定秒数  
5. 自动保存  
6. 返回列表继续，直到全部完成  

右下角有进度浮层，可随时停止。

1. 先安装并启用主脚本 `urppp.user.js`

## 使用

### 打开设置

1. 启用主脚本后，点顶栏 **齿轮**
2. 在设置底部 **辅助插件** 区点击：
   - **登录助手**
   - **评教助手**

弹层位置与主设置窗一致。

### 登录助手配置

需要填写：

- OCR 服务地址（请自建，勿使用不明服务）
- 教务学号 / 密码
- （可选）统一认证账密，或开启“共用账密”
- 是否自动登录、提交延迟

OCR 接口约定：

```http
POST /api/ocr
Content-Type: application/json

{"image":"<base64 不含 data: 前缀>"}
```

期望返回：

```json
{"status":"success","code":"验证码"}
```

### 评教助手配置

建议先配置：

1. 分数上下限  
2. 单选默认池（如 `A` 或 `A,B`）  
3. 特殊题按题配置（如国际周第 7 题）  
4. 多选勾选池  
5. 主观题模板（回车分隔多条，随机选用）  
6. 等待秒数（默认 100）  
7. 是否开启到时自动保存 / 全自动评教  

全自动：

1. 打开教学评估列表页  
2. 设置里点 **启动全自动评教**，或油猴菜单同名项  
3. 保持页面开启，等待逐份完成
