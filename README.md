# UPR++

四川大学 URP 教务系统增强插件。

## 参考项目：SCU URP Assistant 功能记录

SCU URP Assistant（作者：王兆基，2018 年起维护）是针对四川大学综合教务系统的油猴脚本，以下为其已实现的功能清单（仅功能记录，不涉及源码使用）：

### 评教相关
- **一键评教**：自动填充评教表单并提交，适配新旧两版教务系统
- **去除评教等待限制**：移除手动评教页面的 2 分钟等待时间

### 成绩相关
- **绩点/均分增强计算**：在成绩页面提供更详细的 GPA 和加权均分计算结果
- **分项成绩查看**：查看课程各分项得分明细
- **成绩数据分析**：对历史成绩进行统计分析

### 视觉优化
- **beautify 美化样式**：对教务系统页面进行 CSS 层面的视觉美化
- **课表排版优化**：对课程表页面进行重排优化
- **页面提示增强**：增强页面 tooltip 提示信息

### 课程相关
- **教材选择辅助**：辅助教材选定
- **课程信息交流**：课程评价与信息共享

### 培养方案
- **专业授位查询**：查询专业的学位授予信息
- **培养方案查询**：查询专业培养方案详情

### 登录体验
- **恢复免登录选项**：恢复登录页"两周内免登录"复选框

### 创新/大创
- **创新班/大创查询**：查询历届大学生创新创业项目信息

### 辅助功能
- **设置面板**：提供可视化的插件设置界面
- **更新日志**：展示版本更新内容
- **反馈通道**：用户反馈入口
- **用户体验改善计划**：匿名数据收集用于改进
