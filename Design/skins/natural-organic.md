# 自然有机风 · URP 适配摘要

- StyleKit: https://www.stylekit.top/zh/styles/natural-organic
- Registry: ../natural-organic.registry.json
- 官方完成度自评: D 59/100

## 核心理念
大地色、温暖、有机圆角、手工感轻纹理；拒绝冷工业。

## 可映射到 Skin Token
| Token | 目标值 |
|-------|--------|
| radius | 大：卡片 ~2rem，按钮 pill |
| border | stone-200 细线 |
| shadow | 无或极轻 |
| font-display | 衬线标题 |
| bg | 奶油 #faf6f1 |
| primary | 棕/橄榄/鼠尾草 |

## 禁止
冷蓝紫主色、纯黑、尖锐直角、霓虹、重阴影。

## 教务适配注意
- 状态色（成功/警告/危险）要映射到暖色体系
- 可选极轻 noise 纹理（opacity ≤ 0.04）
- 对「消 AI 味」很有效：温度与材质感

## URP 优先级
P1
