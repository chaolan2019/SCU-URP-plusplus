# 极简扁平风 · URP 适配摘要

- StyleKit: https://www.stylekit.top/zh/styles/minimalist-flat
- Registry: ../minimalist-flat.registry.json
- 官方完成度自评: C 69/100

## 核心理念
去掉光影；纯色；留白；用颜色与反色建立层次。

## 可映射到 Skin Token
| Token | 目标值 |
|-------|--------|
| radius | 0 或统一小值（勿混用） |
| border | 2px solid 黑/主色 |
| shadow | none |
| hover | 背景/文字反色 |
| colors | 黑白 + 一枚强调色 |

## 禁止
任何 shadow、渐变、毛玻璃、灰色糊背景当主表面。

## 教务适配注意
- 与当前 Soft Utility 最近，改动最小
- 表格适合硬边线，信息更清晰
- 可做「去 AI 味」的冷硬方案
